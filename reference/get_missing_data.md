# retrieve 'case_id's which have missing values in computationally crucial variables

retrieve 'case_id's which have missing values in computationally crucial
variables

## Usage

``` r
get_missing_data(data)
```

## Arguments

- data:

  data.frame, raw linelist of surveillance cases

## Value

list of column names, number of missing values and the first five
'case_id's for which values are missing
