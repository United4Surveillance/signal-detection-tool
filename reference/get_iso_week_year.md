# Get isoweek and isoyear from a given date

Get isoweek and isoyear from a given date

## Usage

``` r
get_iso_week_year(date)
```

## Arguments

- date:

  character in the format "yyyy-mm-dd"

## Value

list containing a isoweek (integer) and isoyear (integer)

## Examples

``` r
if (FALSE) { # \dontrun{
get_iso_week_year("2020-03-04")
} # }
```
