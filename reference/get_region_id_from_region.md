# Function to get the region_id variable names from the region variables

Function to get the region_id variable names from the region variables

## Usage

``` r
get_region_id_from_region(region)
```

## Arguments

- region:

  character, such as "county", "state"

## Value

character, attaching "\_id" to the region name, i.e. "county_id",
"state_id"
