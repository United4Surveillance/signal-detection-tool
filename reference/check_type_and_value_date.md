# Checking type and values of date variables date variable can be of type character or date

Checking type and values of date variables date variable can be of type
character or date

## Usage

``` r
check_type_and_value_date(data, date_var)
```

## Arguments

- data:

  data.frame, raw linelist of surveillance cases

- date_var:

  character, date variable to check

## Value

list, empty when no errors occured or filled with error messages
