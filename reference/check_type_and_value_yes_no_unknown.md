# Checking type and values of variables which should have yes,no,unknown values

Checking type and values of variables which should have yes,no,unknown
values

## Usage

``` r
check_type_and_value_yes_no_unknown(data, var)
```

## Arguments

- data:

  data.frame, raw linelist of surveillance cases

- var:

  character, variable to check

## Value

list, empty when no errors occured or filled with error messages
